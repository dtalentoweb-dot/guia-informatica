⚙️ Antes de empezar: Guía Técnica y de Navegación

Sigue estos pasos para asegurar una experiencia de aprendizaje perfecta.

🖥️

1. Maximiza tu espacio

Este curso utiliza Libros Interactivos anchos. Para verlo todo perfecto:

  • Oculta el Menú Lateral del Curso: Haz clic en el botón X o ☰.
  • Oculta el Índice del Libro: Dentro de los ejercicios, cierra la columna gris izquierda.
🛠️

2. Herramientas externas

✅ Recomendado: Usa WordReference en otra pestaña.

❌ A evitar: No uses el traductor automático del navegador sobre la página del curso.

🗺️

3. Ruta recomendada

🚀 Orden sugerido:
Aunque la navegación es libre, sigue el orden: Unidad 0 → Unidad 1 → Unidad 2... para asentar las bases.
